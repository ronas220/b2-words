# B2 Words — Английский словарь

Мобильное PWA-приложение для изучения **1335 английских слов уровня B2** с русским переводом, примерами и озвучкой. Работает полностью офлайн после первой загрузки.

## Возможности

- **Интервальные повторения (SRS)** — карточки живут в коробках Лейтнера (10 мин → 1 → 3 → 7 → 21 день). Режим **«План»** собирает очередь на сегодня: просроченные повторы (сначала самые старые) + новые слова в пределах дневного лимита (5/10/15/25). Режим **«Свободно»** — прежний свободный просмотр; ответы «Знаю/Не знаю» в обоих режимах двигают слово по коробкам. «Выучено» = коробка 3+.
- **Дневная цель и серия** — цель ответов в день (10/20/30/50), прогресс «Сегодня N/цель» и огонёк-серия дней с выполненной целью в шапке карточек; празднование с конфетти по завершении плана.
- **Карточки** — 3D-переворот по тапу, свайпы (вправо — «знаю», влево — «не знаю»), клавиатура на десктопе (← / → / пробел), автоозвучка, фильтры (перемешать, буква A–Z, только невыученные), позиция в колоде сохраняется.
- **Список** — поиск по слову и переводу, липкий поиск, быстрый A–Z переход по алфавиту, отметки «выучено» (записываются в SRS).
- **Выбор** — состав слов для режима «План»: чекбоксы по всему словарю с поиском, фильтрами по букве и статусу SRS («Новые» / «В работе» / «Выученные») и масс-действиями «Выбрать/снять видимые». Снятые слова не попадают в очередь плана в карточках и тесте, но их SRS-прогресс сохраняется; «Свободно» и «Список» всегда показывают весь словарь. По умолчанию выбраны все 1335 слов.
- **Спринт** — интенсив на сегодня без дневных лимитов: в карточках и тесте третий сегмент режима. Колода/вопросы строятся по ВСЕМ выбранным во вкладке «Выбор» словам (лимит новых и интервалы игнорируются). В карточках «Знаю» убирает карточку, «Не знаю» возвращает её примерно через 3 карточки — сессия идёт до полного прохождения («Осталось X из N»). В тесте — раунды по 20 вопросов до конца слов, в финале общий счёт и список ошибок (они уходят в 1-ю коробку). Ответы двигают SRS-коробки и идут в дневную цель/серию.
- **Экспорт/импорт** — кнопки «Экспорт»/«Импорт» в карточках (рядом с настройками цели). Экспорт скачивает `b2words-backup-YYYY-MM-DD.json` со всеми данными (SRS, выбор слов, статистика, настройки). Импорт строго валидирует файл (версия, типы, отброс неизвестных слов), заменяет текущие данные после подтверждения и показывает тост с итогом.
- **Тест** — режим **«План»** задаёт вопросы из той же сегодняшней SRS-очереди (сначала повторы, затем новые слова из общего дневного лимита; до 20 вопросов за раунд): верный ответ двигает коробку как «Знаю», неверный — возвращает слово в 1-ю коробку на 10 минут. Режим **«Свободно»** — прежний случайный раунд из 10 вопросов (SRS не трогает, в дневную цель засчитывается). Авто-переход после верного ответа, рекорд, конфетти.
- **Озвучка** — Web Speech API (английские голоса), озвучка слов и примеров.
- **Тёмная тема** — переключатель в шапке, по умолчанию следует системной.
- **Офлайн** — Service Worker кэширует всё: код, данные, шрифты, иконки.
- Прогресс хранится локально (localStorage: SRS-коробки, дневная активность, настройки), ничего не отправляется наружу. Старые отметки «выучено» при первом запуске переносятся в 5-ю коробку.

## Запуск для разработки

```bash
npm install
npm run dev        # http://localhost:3000
```

## Сборка

```bash
npm run build      # продакшен-сборка в dist/
npm run preview    # локальный просмотр сборки
```

## Деплой

`dist/` — обычный статический сайт. Загрузите содержимое папки на любой статический хостинг (Vercel, Netlify, GitHub Pages, nginx и т.д.). Для корректной работы PWA нужен HTTPS (или localhost).

## Установка как приложение (PWA)

- **Android / Chrome**: меню ⋮ → «Установить приложение» / «Добавить на главный экран».
- **iOS / Safari**: «Поделиться» → «На экран "Домой"».

## Данные

Слова вшиты в бандл (`src/data/words.ts`, генерируется из JSON). Структура записи: слово, часть речи, перевод, пример и его перевод.
